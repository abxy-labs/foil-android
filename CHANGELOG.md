# Changelog

## 1.2.0

- Adds the native behavioral raw form/action provenance wire contract.
- Expands Android runtime, app integrity, and anti-tamper signal coverage.
- Keeps public install coordinates unchanged:
  `com.tripwirejs:tripwire-android` and
  `com.tripwirejs:tripwire-android-gms`.

## 1.0.0

Initial public Android SDK package release.

- Primary artifact: `com.tripwirejs:tripwire-android:1.0.0`
- Optional Google helper artifact: `com.tripwirejs:tripwire-android-gms:1.0.0`
- Minimum supported Android version: Android 6.0 Marshmallow / API 23
- Runtime integrity and anti-tamper collection is always enabled
- Public repository contains package metadata and documentation only; protected
  binary AAR artifacts are distributed through Maven Central
