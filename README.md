# Tripwire Android SDK

Install from Maven Central:

```kotlin
dependencies {
    implementation("com.tripwirejs:tripwire-android:1.2.0")
    implementation("com.tripwirejs:tripwire-android-gms:1.2.0")
}
```

This public repository contains the package metadata for the protected binary
SDK. The release artifacts are published to Maven Central as
`com.tripwirejs:tripwire-android:1.2.0` and
`com.tripwirejs:tripwire-android-gms:1.2.0`.
